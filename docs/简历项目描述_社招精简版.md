# 简历项目描述社招精简版

## 项目名称

NVMe SSD 自动化测试与低层验证项目

## 项目描述

自行采购 `Lexar SSD ARES PRO 1TB NVMe SSD` 作为测试对象，基于 Ubuntu 搭建 NVMe SSD 自动化测试项目，并在 Ubuntu 与树莓派环境中完成真实硬件验证，覆盖功能正确性、基础性能、TRIM/Flush、SMART 健康监控及低层命令通路测试。

## 项目职责与成果

1. 独立设计并实现 NVMe SSD 自动化测试方案，使用 Python 开发测试主控脚本，完成设备识别、裸盘 pattern 校验、TRIM/Flush、SMART 对比及测试报告输出。
2. 基于 `fio` 搭建顺序读写、随机读写及混合读写性能冒烟测试，支持按块大小、队列深度、偏移区间及读写比例进行参数化配置。
3. 基于 C 和 Linux `NVME_IOCTL_ADMIN_CMD` 开发 NVMe 低层测试工具，实现 `Identify`、`SMART Log`、`Error Log`、`Flush` 等 Admin 命令验证。
4. 基于 C 开发 `O_DIRECT` 直通 I/O 校验工具，完成对齐检查、裸盘写入/回读及数据一致性验证。
5. 在真实硬件测试中完成多项功能与性能验证，测试结果未发现数据一致性错误、介质错误或新增 SMART 异常，形成可复用的脚本、工具和文档闭环。

## 可选一句话版本

独立完成 NVMe SSD 自动化测试项目建设，自行采购 `Lexar SSD ARES PRO 1TB` 作为测试盘，在 Ubuntu 与树莓派环境中基于 Python、C、`fio`、`nvme-cli` 实现 SSD 功能、性能、TRIM/Flush、SMART 及低层命令验证。
